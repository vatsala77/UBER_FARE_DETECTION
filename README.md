# 🚖 Uber Fare Prediction Dashboard

An interactive ML web app built using **Streamlit** to predict Uber fares.

---

## ✨ Features

* 📊 Interactive dashboard
* 🧹 Data cleaning & feature engineering
* ⚠️ Outlier detection (IQR)
* 🤖 Model training (Linear Regression & Random Forest)
* 📈 Performance metrics (MAE, RMSE, R²)
* 📉 Actual vs Predicted visualization

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas, NumPy
* Scikit-learn
* Plotly

---

## ⚙️ Setup

1. **Clone the repository:**

```bash
git clone https://github.com/vatsala77/uber-fare-prediction.git
cd uber-fare-prediction
```

2. **Install dependencies:**

```bash
pip install streamlit pandas numpy scikit-learn plotly statsmodels
```

3. **Run the application:**

```bash
streamlit run TEST.py
```

---

## 🌐 Open in Browser

```bash
http://localhost:8501
```

---

## 🧠 How to Use

1. Upload dataset (CSV)
2. Go to Cleaning tab → Apply Cleaning
3. Go to Model tab → Select model → Train
4. View results in Performance tab

---

## 📊 Model Insights

* Linear Regression → Underfitting
* Random Forest → Better performance (non-linear patterns)
